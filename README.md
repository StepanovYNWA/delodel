# Генератор счёта на оплату

Онлайн-генератор счёта на оплату для ИП, ООО и самозанятых. Self-contained HTML-файл, работает офлайн после первой загрузки.

**Демо:** https://example.github.io/invoice/

## Что умеет

- Заполнение реквизитов поставщика и покупателя для всех форм юрлиц (ИП, ООО, самозанятый, физлицо)
- Валидация контрольных сумм ИНН, ОГРН/ОГРНИП, расчётного счёта (по 762-П ЦБ РФ)
- QR-код по ГОСТ Р 56042-2014 для оплаты сканированием через банк
- Сумма прописью на русском с правильным склонением
- Экспорт в PDF (формат А4, кириллический шрифт зашит в файл)
- Автозаполнение по ИНН (демо использует egrul.org, в продакшене — DaData)
- Адаптивная вёрстка, работает на телефоне

## Технологии

- Чистый HTML/CSS/JS, без сборки и фреймворков
- jsPDF + AutoTable для генерации PDF
- qrcodejs для QR-кодов
- Шрифт DejaVu Sans (subset с кириллицей) встроен в HTML как base64

## Деплой

Это статический сайт. Достаточно положить `index.html` в любой веб-сервер. Используется на GitHub Pages.
